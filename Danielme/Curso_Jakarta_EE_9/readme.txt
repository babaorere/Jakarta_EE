.- instalacion de software (Linux)

	.- Instalar sdkman: curl -s "https://get.sdkman.io" | bash 
	.- Instalar Java JDK
			.- Recomendado instalada Java version 11 o superior. Verificar las versiones disponibles de JDK: sdk list java
			.- Instalar la version escogida, por ejemplo: sdk install java 11.0.10.hs-adpt
	.- Instalar IntelliJ o Eclipse Enterprise Java Developers	
	.- Instalar Maven
			.- Verificar versiones disponibles con: sdk list maven
			.- Instalar la version escogida, por ejemplo: sdk install maven 3.8.6
			
.- Crear proyecto simple jakarta "Hello"
